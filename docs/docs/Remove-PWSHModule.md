# Remove-PWSHModule

## SYNOPSIS
Remove a module to the config file

## SYNTAX

```
Remove-PWSHModule [-GitHubUserID] <String> [-GitHubToken] <String> [-ListName] <String> [-ModuleName] <String>
 [<CommonParameters>]
```

## DESCRIPTION
Remove a module to the config file

## EXAMPLES

### EXAMPLE 1
```
Remove-PWSHModule -Export HTML -ReportPath C:\temp
```
