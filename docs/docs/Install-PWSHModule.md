# Install-PWSHModule

## SYNOPSIS
Install modules from a config file

## SYNTAX

```
Install-PWSHModule [[-GitHubUserID] <String>] [[-GitHubToken] <String>] [[-ListName] <String>]
 [[-Scope] <String>] [<CommonParameters>]
```

## DESCRIPTION
Install modules from a config file

## EXAMPLES

### EXAMPLE 1
```
Install-PWSHModule -Export HTML -ReportPath C:\temp
```
