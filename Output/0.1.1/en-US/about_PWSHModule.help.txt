
PWSHModule
	 about_PWSHModule
 
SHORT DESCRIPTION
	 Uses a Config file to install and maintain a list of PowerShell Modules

 
NOTES
Functions in this module:
	 Add-PWSHModule -- Add a module to the config file
	 Install-PWSHModule -- Install modules from a config file
	 New-PWSHModuleConfigFile -- Create a new config file.
	 Remove-PWSHModule -- Remove a module to the config file
	 Show-PWSHModule -- List the content of a config file
